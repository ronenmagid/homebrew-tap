# Ronenmagid Tap

## How do I install these formulae?

`brew install ronenmagid/tap/<formula>`

Or `brew tap ronenmagid/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "ronenmagid/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
